反转链表

题目描述

定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

示例:

1
2

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

题目解析

用双指针解决此题，定义2个指针，pre 和 cur。 1.pre指向null，cur指向头结点。 2.cur.next指向pre，实现局部反转。

3.局部反转后，让pre和cur都前进一步，cur = cur.next; pre = cur;

4.循环以上步骤直到cur等于null，返回pre。

public ListNode solution(ListNode head) {
    ListNode pre = null;
    ListNode cur = head;

    while (cur != null) {
        ListNode next = cur.next; // 临时存储next节点
        cur.next = pre; // 局部反转
        pre = cur; // pre前进一步
        cur = next; // cur前进一步

    }
    return pre;
}
