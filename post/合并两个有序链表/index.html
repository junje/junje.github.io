合并两个有序链表

题目描述

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例 1：

1
2

输入：l1 = [1,2,4], l2 = [1,3,4]
输出：[1,1,2,3,4,4]

示例 2：

1
2

输入：l1 = [], l2 = []
输出：[]

示例 3：

1
2

输入：l1 = [], l2 = [0]
输出：[0]

题目解析

此题可以用递归或者循环来解决，但是递归不好理解，所以采用循环方式。

1.定义一个prehead节点，便于返回合并后的链表。 2.定义一个指针pre，遍历的过程中调整pre.next。

3.判断L1和L2的节点值哪个更小，把小的节点放在pre后面，比如L1的更小，pre.next=l1，然后l1前进一步， l1 = l1.next，如果L2小就让pre.next指向L2。

4.pre前进一步。pre=pre.next。

5.循环后，L1和L2有可能会有剩余节点，我们把剩余节点拼在pre后面就可以了，pre.next=剩余节点。

class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
      	// 定义头节点
        ListNode head = new ListNode();
      	// 定义指针 用来遍历
        ListNode pre = head;
        while (l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                pre.next = l1;
                l1 = l1.next;
            } else {
                pre.next = l2;
                l2 = l2.next;
            }
            pre = pre.next;
        }
        pre.next = l1 == null ? l2 : l1;
        return head.next;
    }
}
